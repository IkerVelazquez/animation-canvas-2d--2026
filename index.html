Sobre esta Aplicación

Esta aplicación permite simular diferentes efectos físicos utilizando HTML5 Canvas y JavaScript puro. Experimenta con distintos modos de animación: explosiones dinámicas, caídas de lluvia física, movimiento libre de asteroides y rebotes exagerados. El usuario puede personalizar el tamaño del lienzo y la cantidad de elementos para observar el comportamiento de los objetos con gravedad, colisiones y fricción realista.
